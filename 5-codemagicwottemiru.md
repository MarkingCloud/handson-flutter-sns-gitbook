# 5. Codemagicを使ってみる



{% hint style="success" %}
**work**

* Codemagicへ移動する
* Add application を選択
* Select teamにて「Personal account」を選択 > Next
* Connect repositoryにて「GitHub」を選択 > Next
* Set up applicationにて作成したレポジトリを選択 > 「Flutter App」を選択 > Finish
{% endhint %}
